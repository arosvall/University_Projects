This was the first assignment in an Intro to Artificial Intelligence paper
The assignment was comprised of three parts
1: Using the iris dataset apply KNN to find the nearest K neighbours of an instance and classify it. This achieved an accuracy of up to 97% dependent on the number of neighbours
2: Using the hepatitis dataset create a Decision Tree to classify instances
3: Create a simple perceptron to classify instances after generating the instances for the perceptron. 

This assignment achieved an A grade
